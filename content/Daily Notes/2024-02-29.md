# Homework
```dataview
TABLE course, due, status
FROM "Homework" 
WHERE ( due >= date(2024-02-29) OR status != "Complete" ) AND time.ctime <= date(2024-02-29)
SORT due
```
# Upcoming Tests
```dataview
TABLE course, date
FROM "Tests" 
WHERE date >= date(2024-02-29) and file.ctime < date(2024-02-29)+dur(1 day)
SORT date
```
