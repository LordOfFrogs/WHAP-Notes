# Homework
```dataview
TABLE course, due, status
FROM "Homework" 
WHERE ( due >= date(2024-03-07) OR status != "Complete" ) AND time.ctime <= date(2024-03-07)
SORT due
```
# Upcoming Tests
```dataview
TABLE course, date
FROM "Tests" 
WHERE date >= date(2024-03-07) and file.ctime < date(2024-03-07)+dur(1 day)
SORT date
```
